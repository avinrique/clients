# Vinayaka Consultancy — Client College Pages

Static HTML pages for [Vinayaka Consultancy](https://vinayakaconsultancy.com), providing detailed college admission information, fee structures, placement data, and application guidance.

## Pages

- **RV_College.html** — RV College of Engineering (RVCE), Bangalore
  - Complete admissions guide (KCET, COMEDK, Management quota)
  - Fee structure with channel-wise breakdown
  - Placement statistics and top recruiters
  - Interactive seat distribution chart (Chart.js)
  - FAQ accordion, sticky navigation, responsive design

## Tech Stack

- Vanilla HTML/CSS/JS (single-file, self-contained)
- [Poppins](https://fonts.google.com/specimen/Poppins) (Google Fonts)
- [Font Awesome 6](https://fontawesome.com/) (icons)
- [Chart.js 4](https://www.chartjs.org/) (data visualization)
- No frameworks or build tools required

## Deployment

Each `.html` file is standalone and can be served directly. Upload to any static host or drop into the existing Vinayaka Consultancy web root.
